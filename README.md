# Library Management System (Console-Based)

This is a simple **Library Management System** built using **Java**, designed to manage books in a library using console inputs.

## 🚀 Features
- Add new books  
- Issue books  
- Return books  
- View all available books  
- Search for a book  
- Simple and user-friendly console UI  

## 🛠️ Technologies Used
- Java  
- OOP Concepts (Classes, Objects, Methods)

## 📘 What I Learned
- Building real-world console applications  
- Applying Object-Oriented Programming  
- Writing reusable and clean code  
- Handling user inputs and menu-driven programs  

## 📎 Future Enhancements
- Add database support (MySQL/SQLite)  
- Add GUI using JavaFX or Swing  
- Add user login system
